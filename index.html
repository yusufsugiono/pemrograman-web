Pengantar

Selamat datang! Halaman ini adalah halaman pembuka dari sekumpulan project hasil tugas-tugas mata kuliah dasar pemrograman berbasis web, yang saya dapatkan pada semester 4. Selama satu semester, saya mempelajari mengenai web statis menggunakan HTML, CSS, Javascript, bahkan hingga web dinamis menggunakan PHP dan MySQL Database.

Project yang ada di sini mungkin jelek-jelek dan tidak begitu penting. Namun project ini telah mewarnai perjalanan saya untuk belajar pengembangan website. Tanpa adanya project-project ini, mungkin saya tidak pernah sampai di titik ini. Jadi saya putuskan untuk saya showcase saja disini, barangkali bisa saya gunakan untuk bernostalgia. Syukur-syukur kalau bisa dijadikan bahan referensi buat kalian, hehe. Oh iya, project yang ada di sini mungkin belum semuanya ter-upload. Jadi akan saya tambahkan lagi secara bertahap kedepannya. Selamat berkeliling dan melihat-lihat ya!
