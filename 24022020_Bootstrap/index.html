Mengapa Web Developer?

Rabu, 26 Februari 2020 | Oleh : 180441100060 | Yusuf S.

Saya memilih untuk menekuni tentang Web karena menurut saya pemrograman Web adalah pemrograman yang paling fleksibel, mudah dan murah. Fleksibel yaitu dapat diakses melalui mobile maupun desktop. Kemudian pemrograman Web ini menurut saya tergolong mudah dibandingkan pemrograman lainnya karena tidak terlalu membutuhkan tools yang membebani perangkat. Selain itu, mempelajari Web tidak harus di laptop/PC, cukup dengan smartphone berspesifikasi rendah pun bisa untuk belajar pemrograman Web.

Dengan tiga hal tersebut di atas, maka saya bisa memperkirakan bahwa aplikasi berbasis Web masih bisa diandalkan dan tetap eksis kedepannya. Untuk itulah saya bercita-cita menjadi seorang Web Developer. Apalagi sekarang ini sudah banyak framework yang bisa digunakan serta dapat dikembangkan juga menjadi Progressive Web Apps.
